Partially logged out

Your IP address is no longer automatically granted access to protected resources.

However, since this log on was done using "basic HTTP authentication", your credentials are now cached by your browser. This means that you, or anyone else using this computer, will automatically be reauthenticated simply by browsing to the authentication gateway.

To log out completely you will need to close all browser windows. When this is done, open a new browser window and connect to the authentication gateway. If you are met by a "Logon successful" message, you have not succeeded in clearing the logon cache. Try logging out of your local computer, or rebooting it.

Also, if this is "kiosk" system, for instance an Internet café machine, you should also clear the browser's cache files.
